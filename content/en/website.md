## Sustainability

We take environmental responsibility seriously and have tried to create a website that has minimal environmental impact. We're achieving this by reducing the amount of data that is transferred from the server to you (the total size of our entire website is just over a megabyte!), by creating a static site which doesn't require expensive computation to serve our web pages, and avoiding unnecessary scripts and tracking. Currently, we are exploring the possibility of serving the website from a low-energy server that will be powered by a small solar panel here in Tallahassee.  

## Accessibility

This site conforms to the [Web Content Accessibility Guidelines (WCAG)](https://www.w3.org/WAI/standards-guidelines/wcag/), which defines requirements for designers and developers to improve accessibility for people with disabilities. It defines three levels of conformance: Level A, Level AA, and Level AAA. Based on testing that we have done, we believe that the Leon Environmental Caucus Website is fully conformant with WCAG 2.1 level AA. The guidelines, however, are complex and test results are often inconsistent from one service to another. Please let us know if you find any aspects of the website that cause accessibility issues, so we can make our site as inclusive as possible.
